# Secure-Chat-App
Secure Messenger is a client-server messaging application that implements end-to-end encryption using a hybrid cryptographic scheme combining RSA and AES. The system ensures confidentiality by encrypting messages with AES and securely exchanging encryption keys using RSA public-key cryptography.

The application allows users to register, authenticate, and exchange encrypted messages securely through a central server, which routes messages without accessing plaintext content.
